感谢分享如此优秀的主题！<br>
有点强迫症，在 https://github.com/Bursteretion/halo-theme-fantastic 主题的基础上做了一点点优化：
> 1、全部日期样式改为：yyyy-MM-dd EE <br>
> 2、首页和文章详情页增加了评论量、浏览量、标签，页脚显示靠左 <br>
> 3、主题设置增加 关注我和文章代码主题 <br>
> 4、友链按排序显示 <br>
> 5、摘要如果有代码时，样式会有问题，增加 word-break: break-word;<br>
